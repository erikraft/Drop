How it works

ErikrafT Drop uses WebRTC technology to create direct and secure connections between browsers.

1

Access the Site

Both parties access drop.erikraft.com in their browsers

2

Connect

A connection code is generated and shared between devices

3

Send Files

Drag and drop files to send them directly to the other device

4

Receive

The recipient receives the files securely, ready for download

Technology behind ErikrafT Drop

ErikrafT Drop uses WebRTC (Web Real-Time Communication), an open technology that allows real-time communication directly between browsers. This means that your files are transferred directly between the devices involved, without passing through intermediate servers.

To establish the initial connection, a minimal signaling server is used (which has no access to transferred files) only to help devices find each other on the network. Once connected, all communication occurs directly between them with end-to-end encryption.
