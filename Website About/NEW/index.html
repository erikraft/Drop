Recursos poderosos para compartilhamento de arquivos

Tudo que você precisa para compartilhar arquivos com segurança e privacidade, superando os serviços tradicionais de armazenamento em nuvem

Criptografia de ponta a ponta

Seus arquivos são criptografados antes de sair do seu dispositivo e só podem ser descriptografados pelo destinatário.

Sem servidores intermediários

Conexão direta entre dispositivos usando WebRTC. Seus arquivos nunca passam por servidores de terceiros.

Funciona no navegador

Sem instalação necessária. Basta acessar o site drop.erikraft.com em qualquer navegador moderno e começar a compartilhar.

Suporte a diversos arquivos

Envie fotos, vídeos, documentos, arquivos compactados e qualquer outro tipo de arquivo sem restrições.

Multiplataforma

Funciona em Windows, Linux, macOS, ChromeOS, Android e iOS. Qualquer dispositivo com um navegador moderno.

Interface arrastar e soltar

Interface intuitiva que permite enviar arquivos simplesmente arrastando e soltando na janela do navegador.

Compartilhe com qualquer pessoa, em qualquer lugar

Envie arquivos para outras cidades ou países, mesmo em redes Wi-Fi diferentes, Para fazer isto é só Conectar os dispositivos com Emparelhamento seguro ou usar as Salas públicas temporárias para grupos.

Como funciona

O ErikrafT Drop utiliza tecnologia WebRTC para criar conexões diretas e seguras entre navegadores

1

Acesse o site

Ambas as partes acessam drop.erikraft.com em seus navegadores

2

Conecte-se

Um código de conexão é gerado e compartilhado entre os dispositivos

3

Envie arquivos

Arraste e solte os arquivos para enviá-los diretamente ao outro dispositivo

4

Receba

O destinatário recebe os arquivos com segurança, prontos para download

Tecnologia por trás do ErikrafT Drop

O ErikrafT Drop utiliza WebRTC (Web Real-Time Communication), uma tecnologia aberta que permite comunicação em tempo real diretamente entre navegadores. Isso significa que seus arquivos são transferidos diretamente entre os dispositivos envolvidos, sem passar por servidores intermediários.

Para estabelecer a conexão inicial, é usado um servidor de sinalização mínimo (que não tem acesso aos arquivos transferidos) apenas para ajudar os dispositivos a se encontrarem na rede. Uma vez conectados, toda comunicação ocorre diretamente entre eles com criptografia de ponta a ponta.
